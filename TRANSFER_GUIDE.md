# ラボいきたい！ 移行ガイド

このZIPには、ソースコード、研究室・カード・ランタイムデータ、ロゴなどの公開アセット、仕様書、テスト、ビルド済みファイルが含まれています。

安全のため、APIキー等を含む `.env` と、再生成可能で容量の大きい `node_modules` は含まれていません。

## 必要なもの

- Node.js 20以上（LTS推奨）
- npm

## 起動方法

1. ZIPを展開します。
2. 展開したフォルダでターミナルを開きます。
3. 依存ライブラリをインストールします。

   ```bash
   npm install
   ```

4. `.env.example` を複製して `.env` という名前に変更します。
5. 必要に応じて `.env` の `OPENAI_API_KEY` / `GEMINI_API_KEY` と既定の `AI_MODEL` を設定します。キーはフロントエンドへ露出しません。
6. 開発サーバーを起動します。

   ```bash
   npm run dev
   ```

7. ブラウザで `.env` の `PORT` に対応する `/search`（例：`http://localhost:3008/search`）を開きます。

AIキーを設定しない場合も、ローカルデータを使う基本的な閲覧・検索画面は起動できます。モデルはPCの左レール、モバイルの「メニュー」から選択でき、新しい生成に適用されます。

## 本番ビルド

```bash
npm run build
npm run start
```

ポートを変更する場合は `.env` の `PORT` を変更してください。
