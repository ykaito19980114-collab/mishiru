# visual-audit

## 現在の視覚的特徴

蛍光イエロー、濃いブルー、白いカード、太いロゴ、問いを中心にしたカード構造が強い。研究室検索を堅く見せず、発見感を作れている。

## 維持すべき要素

- 蛍光イエローと濃いブルー
- 白カードのコントラスト
- 既存ロゴ
- 気になる/保存するの軽い反応
- 問いとして研究室を読む構成

## 改善すべき要素

- デスクトップで中央カラムが細くなりすぎる画面がある
- 保存・検索・問いのつながりがまだ弱い
- カード種別が研究室に寄りすぎている
- 蛍光イエローを大きな余白ではなく、意味のある背景面として使う必要がある

## 今回の改善方針

`max-width: 1240px` を基準に、メイン列とコンテキスト列を持つ画面を増やす。問い・研究ルート・学会/ジャーナル候補を淡い青/淡い黄色の面で補助表示する。
